## `jj`의 Git push용 브랜치 이름 개선

- `jj`([Jujutsu](https://docs.jj-vcs.dev/latest/))는 Git 호환 버전 관리 시스템이며, 여러 이유로 **익명 브랜치** 사용을 기대하고 권장함
- Git 저장소로 push할 때는 익명 브랜치에도 이름이 필요하며, `jj`에서는 이를 **bookmark**, Git에서는 branch라고 부름
- `jj git push --change xyz`는 ID가 `xyz`인 리비전을 Git 브랜치 `push-xyz`로 push함
- 기본값은 `jj` CLI가 자주 표시하고 사용하는 change ID를 강조한다는 점에서 자연스럽지만, GitHub 웹사이트의 브랜치 목록에서는 `push-xyz`만으로 어떤 작업인지 기억하기 어려움

## 설정 변경 방식

- [template alias](https://docs.jj-vcs.dev/latest/templates/#configuration) `slugify()`를 새로 만들고, `git_push_bookmark` 템플릿이 이를 사용하도록 바꿈

```
[template-aliases]
"slugify(str)" = '''
truncate_end(
65,
str.first_line()
.replace(regex:'[^[[:alnum:]].]', '-')
.replace(regex:'-{2,}', '-')
.replace(regex:'\.{2,}', '.')
.replace(regex:'(^-+|-+$)', '')
.lower()
)
'''
[templates]
git_push_bookmark = 'slugify(description) ++ "/" ++ change_id.short()'
```

- `slugify()`는 변경 설명의 첫 줄을 사용해 짧은 [slug](https://en.wikipedia.org/wiki/Clean_URL#Slug) 형태 이름을 만들고, 끝에 짧은 change ID를 붙임
- 예를 들어 `jj git push --change ozkspkuyzpwu`를 실행하면 `add-note-about-jj-bookmark-templates/ozkspkuyzpwu`가 생성됨
- 이 이름은 더 읽기 쉬우면서도 `jj` CLI에 표시되는 **리비전 ID**와의 연결을 유지함
- 다른 사람과 공유하는 저장소에 push한다면, 자신의 브랜치를 별도 namespace 아래에 두도록 템플릿을 바꿀 수 있음

```
[templates]
git_push_bookmark = '"ddbeck/" ++ slugify(description) ++ "/" ++ change_id.short()'
```

- Git 브랜치 이름이 항상 안전한지는 보장되지 않음
- Git의 유효한 브랜치 이름 규칙은 [복잡](https://git-scm.com/docs/git-check-ref-format#_description)하며, 템플릿이 유효하지 않은 이름을 만들면 오류가 발생하고 bookmark를 수동으로 만들어야 함

- CI/CD 파이프라인에서 Docker socket 노출과 EC2 metadata 접근을 감사하고, IMDSv2 hop limit 제한을 고려해야 함
- [Package Manager Guard (pmg)](https://github.com/safedep/pmg)는 `preinstall` 실행 전에 패키지를 threat intelligence와 대조하는 오픈소스 설치 프록시임
- [dependency cooldown](https://github.com/safedep/pmg/blob/main/docs/dependency-cooldown.md)은 설정 가능한 시간 창 안에 배포된 버전을 거부해, semver 범위가 새 악성 릴리스로 해석되는 급격한 배포 파동을 줄일 수 있음
- [vet](https://github.com/safedep/vet)은 예상치 못한 `preinstall` 훅, 크기 급증, maintainer 변경 같은 비정상 패키지 업데이트를 CI/CD 파이프라인에 도달하기 전에 탐지할 수 있음
- 단일 계정 아래 547개 패키지, 한 세션에서 무기화된 314개 이상 패키지라는 영향 범위는 npm 신뢰 모델의 구조적 약점을 드러냄

## FiveThirtyEight의 시작과 성장

- ### Daily Kos에서 독립한 선거판 Moneyball

  - FiveThirtyEight.com은 2008년 3월 진보 성향 블로그 Daily Kos의 인기 연재에서 파생돼 “poblano”라는 가명으로 시작됨
  - Silver는 2004년 기업 컨설팅 일을 그만둔 뒤 [PECOTA](https://en.wikipedia.org/wiki/PECOTA) 같은 통계 모델을 Baseball Prospectus용으로 만들었고, Baseball Prospectus는 구독 기반 모델을 일찍 채택한 매체였음
  - “선거를 위한 Moneyball”이라는 설명은 자연스러웠지만, 2008년 FiveThirtyEight.com이 바이럴 히트가 될 정도까지는 예상하지 못함
  - 2008년 대선에는 Barack Obama 등 주목도 높은 인물이 있었고, 당시 디지털 광고 단가가 비교적 건강해 사이트 광고 수익으로 임대료를 감당할 수 있었음
- ### New York Times 시기

  - FiveThirtyEight는 2008년 중반 실명이 공개됐고, Obama가 John McCain을 이길 것이라는 모델의 강한 예측이 맞아떨어진 뒤 언론의 큰 관심을 받음
  - NYT 계약은 Silver가 직원이 아닌 계약자라 다른 수입원을 추구할 수 있고, 퇴사 뒤 모델과 기타 지식재산권(IP) 소유권을 되찾는 **깔끔한 구조**였음
  - 2012년 선거 예측은 50개 주를 모두 맞히는 큰 성공을 거뒀고, 모델 내부 논리상 그 확률은 약 3%였음
  - NYT는 Silver 재직 중 디지털 페이월을 시작했고, 2012년 4분기 구독 수익이 [크게 증가](https://www.bbc.com/news/business-21373596)했으며, Obama-Romney 선거와 시기가 겹침
  - Silver와 변호사는 NYT에 독점 협상 기간을 줬지만, 당시 NYT는 내부 파벌이 강했고 FiveThirtyEight에는 지지자와 비판자가 모두 있었음
- ### ESPN/Disney 선택

  - 최종적으로 ESPN과 계약했으며, 당시 ESPN은 케이블 권리료 기반의 안정적 수익을 가진 최고의 비즈니스라는 자기 인식이 강했음
  - 당시 ESPN 사장 John Skipper는 Grantland 같은 프리미엄 제품에 관심이 있었고, Grantland는 Disney 안에서 FiveThirtyEight의 선례로 작동함
  - 진지하게 검토한 다른 제안은 NBC News, Bloomberg, NYT에서 왔고, Bloomberg와 NYT는 프리미엄 구독 상품을 파는 사업이라는 점에서 FiveThirtyEight/Silver Bulletin과 더 가까운 구조였음
  - ESPN과 NBC는 주로 TV 네트워크였고, Silver는 TV 출연을 좋아하지 않았으며 본인에게 가장 약한 매체라고 봄

## Disney 안에서 어긋난 인센티브

- Skipper는 FiveThirtyEight가 돈을 벌지 않아도 된다는 취지를 비교적 명확히 했고, FiveThirtyEight는 Grantland처럼 ESPN의 큰 사업에서 **장식품**이자 Disney 손익계산서의 반올림 오차에 가까웠음
- Disney는 테마파크, Marvel 영화, NFL 권리 계약처럼 거대한 베팅을 하는 **매크로 기업**인 반면, FiveThirtyEight는 명확히 틈새 브랜드였음
- 이런 구조는 상사의 호의에 지나치게 의존하게 만들었고, Skipper는 2017년 [갈취 스캔들](https://www.nytimes.com/2018/03/15/sports/john-skipper-cocaine-espn.html)로 ESPN을 갑자기 떠남
- FiveThirtyEight는 2014년 3월 Disney/ESPN 아래에서 재출시됐고, 초반 평가는 [대체로 나빴음](https://talkingpointsmemo.com/livewire/paul-krugman-nate-silver-criticism)
- 직원 수를 너무 빠르게 늘렸고, 양을 지나치게 중시했으며, **핵심 제품**에는 충분히 집중하지 못함
- 가장 큰 오류는 FiveThirtyEight를 지속 가능한 사업으로 만들 방법을 거의 아무도 고민하지 않았다는 점임
- 전담 사업 담당자, 제품 담당자, 경제적 지속 가능성에 책임지는 인력이 사실상 없었고, 상업적 제품으로서의 근육과 인프라를 만들지 못함
- 그럼에도 FiveThirtyEight는 매우 인기 있는 웹사이트가 됐고, 2016년 선거 예측은 Chartbeat 기준 [인터넷에서 가장 “참여도 높은” 기능](https://www.poynter.org/tech-tools/2017/the-best-read-story-of-2016-take-a-bow-nate-silver/)이었으며 팟캐스트는 회당 수십만 다운로드를 기록함

## Grantland 폐쇄와 2016년 선거의 충격

- 2015년 Bill Simmons가 Grantland에서 [퇴출](https://www.nytimes.com/2015/05/09/sports/bill-simmons-and-espn-are-parting-ways.html)된다는 소식은 FiveThirtyEight에 불길한 선례였음
- Grantland는 Bill Simmons 이후 [짧고 혼란스러운 시기](https://www.nytimes.com/2015/10/31/sports/grantland-shut-down-by-espn.html)를 거쳐 2015년 말 완전히 폐쇄됨
- Grantland는 ESPN 안에서 FiveThirtyEight와 같은 하위 조직에 있었고, 상당수 고위 관리자를 공유했음
- 2016년 공화당 경선 초반 Donald Trump의 가능성을 낮게 본 판단은 Silver 경력에서 [가장 큰 분석 오류](https://web.archive.org/web/20160527093831/https://fivethirtyeight.com/features/how-i-acted-like-a-pundit-and-screwed-up-on-donald-trump/)로 남음

- 인터넷 여론은 FiveThirtyEight의 2016년 본선 예측을 그렇게 받아들이지 않았고, 선거일 밤과 그 이후는 어려운 시기로 남음

## 매각 시도와 거절된 페이월

- 결국 FiveThirtyEight는 Disney 안에 다시 남았고, ABC News는 ESPN보다 명목상 더 나은 궁합이었지만 수익성 있고 지속 가능한 자산으로 만들기 위한 투자는 여전히 부족했음

- 2020년 COVID 팬데믹 때 FiveThirtyEight는 Disney 공식 지침보다 하루이틀 앞서 재택근무를 시작했지만, 이후 뉴스룸은 예전과 같아지지 않음
- 이 시기 미디어 환경은 원격근무 전환, 명시적으로 보수 성향이 아닌 매체들이 겪은 정치적 [“reckoning”](https://www.natesilver.net/p/the-long-strange-political-shadow), 선거의 압박이 겹쳐 매우 어려웠음
- 세상이 서서히 정상으로 돌아가도 FiveThirtyEight는 회복되지 않았고, Clare Malone 같은 핵심 인력이 대체 계획 없이 해고됨
- 2018년 ABC News와 맺은 새 계약은 명목상 5년이었지만, 3.5년 뒤인 2021년 12월 상호 중도 해지 조항이 있었음
- Silver가 원한 구조는 Disney에 대한 책임을 줄이고 적절한 보수 삭감을 받는 대신 독점성을 없애는 것이었음
- 구체적으로 2022년 모델 라이선스, 정해진 수의 칼럼과 TV 출연은 제공하되, 관리 책임에서 벗어나 멘토·창업자 역할을 하고 Substack 독자를 만들 자유를 원함
- ABC는 이에 사실상 응답하지 않았고, 반대 제안이나 진지한 논의도 없었음
- Disney는 계약 지속 여부 결정 마감시한을 반복적으로 미뤘고, 결국 최종 마감일에도 해지 통보를 보내지 않음
- 이로써 계약은 2022년 중간선거까지 이어졌지만, Silver는 명백한 레임덕 상태가 됐고, 계약상 허용된 [책 작업](https://www.natesilver.net/p/welcome-to-the-river)에 주로 집중함
- 2023년 4월 ABC News 감원 소문 속에서 FiveThirtyEight는 깊고 불규칙한 감원을 겪음
- Silver는 해고 대상은 아니었지만, 계약이 두 달밖에 남지 않았고 직원이 크게 줄어든 상황에서 새 계약을 제안받아도 수락하지 않았을 것으로 봄
- ABC News가 선거 모델에 대한 권리가 더 이상 없다는 점을 충분히 이해했는지는 불확실하며, 선거 모델 라이선스는 Silver의 계약과 함께 만료됨
- Disney는 스포츠 모델의 경우 Silver 퇴사 당시 상태로 계속 게시할 수 있는 비독점 라이선스를 보유했지만, 전체 스포츠 팀을 해고하고 게시를 중단함
- ABC는 이후 G. Elliott Morris를 새 모델 담당자로 채용함
- FiveThirtyEight는 [ABCNews.com](http://abcnews.com) 전체보다 더 많은 웹 트래픽을 만드는 경우가 있었지만, 스포츠 섹션도 사라지고 브랜드는 계속 감가상각되고 있었음

## 2024년 모델 논란과 Silver Bulletin

- ABC의 새 2024년 선거 모델은 Joe Biden의 토론 참패 뒤에도 Trump와 거의 동률로 표시돼 [말 그대로 고장난 상태](https://www.natesilver.net/p/why-i-dont-buy-538s-new-election)였다고 평가됨
- 원인으로는 Morris의 모델 설계가 지나치게 복잡했을 가능성이 있고, 선거 모델 설계 자체가 어렵다는 단서가 붙음
- Biden이 후보에서 물러난 뒤 모델은 한 달 넘게 내려갔고, Kamala Harris의 “Brat Summer” 기간 대부분을 놓침
- 새로 디버그된 버전은 더 일찍 준비됐던 것으로 이해되지만, ABC News PR은 모델에 대한 대중 인식에 [매우 민감](https://www.semafor.com/article/07/18/2024/the-only-place-biden-is-winning-538s-election-model)했고, 과거 FiveThirtyEight가 알려졌던 투명성과 달리 모호하게만 소통함
- Silver Bulletin은 예상보다 많은 지지를 받았고, Silver는 책 투어 기간 여러 매체에 등장함
- 이는 구독 기반 모델이 올바른 선택이었다는 점을 뒷받침함
- Silver Bulletin은 FiveThirtyEight의 인기 요소 일부를 재구성하고 있으며, [선거 모델](https://www.natesilver.net/p/nate-silver-2024-president-election-polls-model), [여론조사 평균](https://www.natesilver.net/p/trump-approval-ratings-nate-silver-bulletin), 스포츠 모델 [PELE](https://www.natesilver.net/p/pele-international-football-rankings-soccer-ratings-projections), [ELWAY](https://www.natesilver.net/p/elway-nfl-ratings-projections-playoff-odds), [COOPER](https://www.natesilver.net/p/cooper-mens-ncaa-basketball-power-ratings)를 다시 내놓고 있음
- Silver Bulletin은 FiveThirtyEight 전체를 재현하려는 시도가 아니며, 작은 팀 구조가 글쓰기와 모델 제작 같은 **창의적 작업**에 더 많은 시간을 줌
- ABC는 Disney 데뷔 11년 뒤인 2025년 3월 FiveThirtyEight를 [완전히 폐쇄](https://www.natesilver.net/p/a-few-words-about-fivethirtyeight)함

## 남은 IP 협상 실패와 방치

- Silver는 1~2년 전 에이전트를 통해 Disney에 남은 IP 인수를 제안함
- 본인이 논리적으로 가장 높은 가격을 낼 가능성이 있으며, 남은 사이트가 방치될수록 가치가 빠르게 떨어진다고 봄
- 최소한 아카이브를 복구하고 Silver Bulletin으로 가는 눈에 띄는 링크를 넣는 방안이 가능했음
- Disney 측은 ABC가 FiveThirtyEight 관리에 대한 Silver의 공개 비판에 짜증을 냈다는 이유로 사실상 거절함
- [Hanlon’s Razor](https://en.wikipedia.org/wiki/Hanlon%27s_razor)는 악의보다 어리석음으로 설명 가능한 일을 악의로 돌리지 말라는 원칙이지만, ABC의 경우 어느 쪽이 더 맞는지 판단하기 어렵다고 봄
- Disney 재직 후반부에는 선거일 밤 방송에서 Silver를 크게 활용한 것 외에는 FiveThirtyEight 관련 결정에 거의 아무 노력도 들이지 않는 것처럼 느껴짐
- FiveThirtyEight는 편집, 교정, 그래픽에 많은 공을 들여 외부에는 매끄러운 조직처럼 보였지만, 내부에는 데이터 저널리즘의 의미를 둘러싼 직원 간 갈등, 창업자와 직원 간 갈등, 뉴스 사이클과 마감의 충돌, Disney와의 긴장이 계속 있었음
- Disney와의 관계는 영화 장면처럼 격렬하기보다는 대부분 **방치**에 가까웠음
- Silver가 퇴사한 뒤 Disney HR은 10년 “castmember” 근속을 기념하는 Mickey Mouse 명패를 보냈지만, 이름을 “Nataniel”로 잘못 적음
